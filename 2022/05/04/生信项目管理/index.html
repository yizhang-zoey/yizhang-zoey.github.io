    1. 备份

    1.1 原始数据备份

    保险以及方便起见，三个备份（狡兔三窟><）：

    • 云盘

    • 服务器

    • 工作电脑或者移动硬盘

    1.2 项目备份

    • 服务器：工作场景
    • 工作电脑或者移动硬盘：定期归档

    2. README 自述文件

    建议项目的一级目录、一级子目录下，有一个 README 文件

    1. 当前目录的作用
    2. 数据：来源，下载方式等信息
    3. 工作流程/数据处理流程
    4. 软件：版本号、安装方法、使用参数
