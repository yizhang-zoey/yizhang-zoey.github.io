的问题，例如分类。

为了日常顺畅地使用这个工具，以及当MLDL领域有新的突破工作后及时用到生物问题中，系统捋一遍MLDL是很有必要的。

For Beginner

对 beginner，集中时间快速完整地捋完一本简易的教材很有必要——反过来也就是说，如果发现某本教材大多数内容尚看不懂，及时放下，之后再战！

  • 集中时间：建议3-5天内，如果有一点基础，本人亲身体验1天也可以
  • 简易的教材：篇幅短、覆盖面广、侧重实践而非理论

Read List

入门

  1. 《AI制胜机器学习极简入门》
    已阅，建议2-3d内快速集中看完
    本书用语清新自然，blog风格，有些语句有“借鉴”其他人的blog的嫌疑，anyway本人亲自完整看过一遍，用起来还不错。
    主要内容是传统的ML算法，个人建议在DL前系统遛一遍传统ML方法，e.g. kNN, k-means, 决策树, 随机森林, SVM等等。
  2. 《Python机器学习基础教程》Introduction to Machine Learning with Python
    已阅，建议1w内快速集中看完
    在第一本的基础上，相对轻松。有些和第一本内容重复，但是稍微深入和细致一些，不必跳过，重新温习一下
  3. 《动手学深度学习》Dive into deep learning Website
    本书全面开源！开源精神 yyds!
    本书作者之一，李沐，在MLDL教学上挺有名，b站的个人账号发布一些MLDL教学、论文解读视频。
  4. 李宏毅《机器学习》2021 Spring Website
    适合作为系统的入门课程，作业一定要做。
    需要有一定Python和PyTorch基础，否则第一个作业就会很痛苦（别问我为啥知道）。
  5. 《深度学习入门》Deep Learning from Scratch Website
    what I cannot create, I do not understand.
    这本书用numpy手把手带读者实现了神经网络的前向传播、反向传播，语言严谨但不啰嗦，一下子点透了我看其它视频/书/资料后仍感觉模糊的地方
    1.1 一定要动手打一遍代码，尤其是第四章、第五章
    1.2 第四章第五章相对难一
    极力推荐！
  6. 《Python神经网络编程》Make Your Own Neural Network Website
  7. 《机器学习实战》Hands-on Machine Learning with Scikit-Learn, Keras, and TensorFlow Website
